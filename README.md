# Paws Rescue Center

Paws Rescue Center is a full-stack Flask web application for an animal rescue organization. It allows visitors to discover adoptable pets, create an account, and submit adoption applications.

The project is being developed as a professional portfolio application using production-minded organization, database design, validation, authentication, testing, and Git workflows.

## Project Goal

The goal is to create a welcoming and reliable platform that helps rescue animals connect with potential adopters while giving rescue staff the tools needed to manage pets and adoption applications.

## Planned Features

### Visitors

- View the rescue center homepage
- Browse animals available for adoption
- View detailed animal profiles
- Learn about the rescue organization

### Registered Users

- Create an account
- Log in and log out
- Submit adoption applications
- View their submitted applications

### Administrators

- Add new animals
- Edit animal information
- Update adoption status
- Remove animal records safely
- Review adoption applications

## Technical Goals

The application will demonstrate:

- Flask routing
- Jinja template inheritance
- Responsive page layouts
- Form creation and validation
- User authentication
- SQLAlchemy models and relationships
- Database migrations
- CRUD operations
- Authorization and protected routes
- Error handling
- Automated testing
- Professional Git and GitHub workflows

## Design Direction

The website should feel:

- Warm and welcoming
- Clean and trustworthy
- Easy to navigate
- Accessible
- Responsive on desktop and mobile devices
